The VOSpace Filesystem Project.

VOSpace is an IVOA standard based protocol for interactions with Astronomical Data Repositories.

cadcVOFS is a project lead by the Canadian Astronomy Data Centre to provide a Python package (vos) for interacting with the a VOSpace standards compliant data repository. The vos package also includes a FUSE based filesystem, enabling a VOSpace to appear as a files system on the users desktop top. Works on Linux and OS-X.

$ pip install vos
$ mountvofs vos://cadc.nrc.ca!vospace/yourVOSpace /mount_point
$ ls /mount_point

This package is still in beta.
