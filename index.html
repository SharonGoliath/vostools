The VOSpace Filesystem Project.

VOSpace is an IVOA standard based protocol for interactions with Astronomical Data Repositories.

CANFAR/VOS is a project lead by the Canadian Astronomy Data Centre to provide a Python package (vos) for interacting with the a VOSpace standards compliant data repository. The vos package also includes a script (mountvofs) that establishes a FUSE based filesystem view of the CANFAR VOSpace. mountvofs enables VOSpace to appear as a filesystem on the users desktop. Works on Linux and OS-X.

$ pip install vos
$ mountvofs --vospace vos://cadc.nrc.ca!vospace/yourVOSpace --mountpoint /root/mount/point
$ ls /root/mount/point

This package is still in beta.
